# JobPortal Admin Panel

A complete, responsive admin panel for a job portal built with **React + Vite + Tailwind CSS**.

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🔐 Demo Login
- **Email:** admin@jobportal.com
- **Password:** admin123

## 📁 Pages
| Page | Route | Description |
|------|-------|-------------|
| Login | `/login` | Auth screen with demo credentials |
| Dashboard | `/dashboard` | Stats, charts, activity feed |
| Job Listings | `/jobs` | Table + grid view, filter, post jobs |
| Applications | `/applications` | Candidate pipeline with status tabs |
| Companies | `/companies` | Manage & verify companies |
| Candidates | `/candidates` | Browse candidate profiles |
| Analytics | `/analytics` | Recharts: funnel, trend, breakdown |
| Notifications | `/notifications` | Read/unread notification center |
| Settings | `/settings` | Profile, security, notification prefs |

## 🛠 Tech Stack
- React 18 + Vite
- Tailwind CSS v3
- React Router v6
- Recharts
- Lucide React Icons
